# [1978] 소수 찾기
---

💡 주어진 수 N개 중에서 소수가 몇 개인지 찾아서 출력하는 프로그램을 작성하시오.


# 풀이

## 소수판별법

1. N까지 해당하는 모든 수를 나눠서 나머지가 0이 되면 소수가 아님. 가장 원초적인 방법.
2. N의 절반까지만 확인하는 판별법. 약수의 중간값을 이용한 원리.
3. \sqrt{N} 까지만 확인하는 판별법. 2번과 비슷함. 중간값이 \sqrt{N}을 못넘기는 원리.

이 중 나는 2번의 방법을 씀.

### 구현
```c
#include <stdio.h>

int prime(int n){
        for(int i = 2;i<=n/2;i++){
                if(n%i==0)return 0;
        }
        return 1;
}

int main (){
        int n;
        scanf("%d",&n);
        int result=0;
        for(int i = 0;i<n;i++){
                int t;
                scanf("%d",&t);
                if(t == 1){
                        continue;
                }
                if(prime(t))result++;
        }
        printf("%d",result);
}
```
